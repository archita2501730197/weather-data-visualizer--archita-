# Weather Data Visualizer

# Dataset

The dataset contains daily weather observations (Temperature, Rainfall, Humidity) from November to December 2024.

# Tools Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- VS Code

# Project Description

This project analyzes weather data to identify patterns and trends. It includes:

- Data cleaning and preprocessing
- Statistical analysis
- Visualizations (line chart, bar chart, scatter plot, combined chart)
- Monthly aggregation of data
- Exporting cleaned datasets and reports

# Results

- Cleaned dataset: `data/weather_data.csv`
- Monthly summary: `data/monthly_summary.csv`
- Plots saved in `images/`
- Summary report: `report.md`

# Insights:

- November was rainier than December
- Maximum temperatures slightly decreased in December
- Humidity correlates slightly with rainfall
