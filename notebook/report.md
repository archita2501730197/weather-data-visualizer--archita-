# Weather Data Visualizer — Report

# Dataset Description

The dataset contains daily weather observations including:

- Date
- Maximum Temperature
- Minimum Temperature
- Rainfall
- Humidity

The data covers November to December 2024.

# Key Insights

# 1. Temperature Trends

- Maximum temperature ranged between 25.2°C and 27.3°C.
- Minimum temperature ranged between 8.3°C and 11.6°C.
- Daily temperature trend shows a slight decrease in December.

# 2. Rainfall

- Total rainfall in November: 8 mm
- Total rainfall in December: 2 mm
- November was rainier than December.

# 3. Humidity

- Average humidity ranged from 57% to 65%.
- Humidity is slightly higher on rainy days.

# 4. Correlation

- Higher temperature days generally have slightly lower humidity.
